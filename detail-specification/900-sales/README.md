##900 - Sales

Consists of Multiple Sales Bundles

* CoreBundle (Basic Sale Functionality needed by all Sales Bundles)
* OrderBundle (Handling Orders)
* InvoiceBundle (Handling Invoices)
* ShippingBundle
* QuotationBundle
