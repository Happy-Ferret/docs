####FR-1401 Multi-formats

**Definition:**

The system shall provide a means of delivering correct data formats over a defined URL with a specific data type extension.

**Specification:**

1. Following data formats shall be supported: HTML, XML and JSON
1. RSS as a extension of HMTL should be supported for smart pages


####![Alpha](https://raw.github.com/massiveart/sulu-docs/master/system-requirements/images/alpha.png)FR-1402 Themes

The system shall provide facilities to manage multiple themes covering all included components, like Java Script, templates, style sheets, assets etc.
