####FR-1451 Migration

The system shall provide a migration concept for import legacy data of ZOOLU 1.0.

####![Alpha](https://raw.github.com/massiveart/sulu-docs/master/system-requirements/images/alpha.png)FR-1452 General API

**Definition:**

The system shall provide a REST based API for all implemented entities.

**Specification:**

1. The API shall consider the user authentication to meet the security requirements
1. Data delivery shall be based on the security settings
